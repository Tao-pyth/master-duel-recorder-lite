# Roadmap

## v0.1.0: 土台作成

- Python パッケージ構成を作る
- `user_data/` の保存先を定義する
- 設計ドキュメントを追加する
- 最小CLIで実行確認できるようにする

## v0.2.0: 最小録画検証

- FFmpeg の存在確認を行う
- 短時間の画面録画を保存する
- 録画失敗時のログを残す

## v0.3.0: 検出処理の接続

- 録画中フレームを解析できる形にする
- 状態遷移を使って録画開始と停止の候補を判定する
- OCRは補助的に扱う

## v0.4.0: 履歴とキュー

- SQLite に録画履歴を保存する
- アップロード準備キューを作る
- 再起動後の復旧処理を追加する
