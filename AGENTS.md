# AGENTS.md

このリポジトリは Python 中心の録画支援ツールとして開発します。説明は日本語で、初心者エンジニアにも意図が伝わるようにしてください。

## 重要方針

- OBS Plugin と OBS WebSocket に依存する実装を追加しない
- 録画、検出、保存、アップロード準備の責務を Python 側で分離する
- `user_data/` 配下の実行時データを更新や実装変更で壊さない
- SQLite、キュー、認証情報を扱う場合は復旧安全性を優先する
- ゲーム素材、スクリーンショット、抽出テンプレートをコミットしない

## コード方針

- 読みやすさを優先する
- 不要な抽象化を避ける
- 初心者向けに必要なコメントだけを短く書く
- DB処理、録画処理、アップロード処理を混ぜない

## ドキュメント方針

仕様変更や設計判断は `docs/` に残してください。特に録画方式、保存先、復旧処理、ユーザー設定に関わる変更は、実装前に短い設計メモを追加します。
